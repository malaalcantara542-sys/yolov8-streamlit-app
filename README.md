# 🎥Live Camera Object Detection  – Object Detection System

A real-time **AI-powered object detection web application** built using **Streamlit** and **YOLOv8**.  
This project demonstrates how artificial intelligence can be used for **live image analysis, object recognition, and smart vision systems**.

---

## 🚀 Features

- 📷 Live Camera Object Detection
- 🖼️ Image Upload Detection
- 🤖 Powered by YOLOv8 (Ultralytics)
- ⚡ Fast real-time inference
- 🎯 Object counting system
- 🎨 Semi-professional UI (Streamlit custom design)
- 🧠 AI-based vision processing

---

## 🛠️ Technologies Used

- Python 🐍
- Streamlit 🌐
- YOLOv8 (Ultralytics) 🤖
- OpenCV 👁️
- NumPy 🔢
- PIL (Python Imaging Library) 🖼️
##📸 How It Works
Open the app in browser
Choose mode:
Live Camera 📷
Upload Image 🖼️
The YOLOv8 model detects objects in real-time
Results are displayed side-by-side (Original vs Processed)
